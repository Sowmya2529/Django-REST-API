Bank Rest API


To get branch details,the url is - 'https://credicxointerntask.herokuapp.com/branchdetails/(ifsc_code)'

Sample - Click this link to view the details of branch with ifsc code - AKJB0000009


To get the details of all the branches of a bank in a given city, the url is 'https://credicxointerntask.herokuapp.com/branchdetails/(bank_name)/(city_name)'

Sample - Click this link to view all the branch details of ABHYUDAYA COOPERATIVE BANK LIMITED in Mumbai


This API works for all Ifsc_codes, bank_names and cities given in the database

Ensure that the url paramaters provided by you are valid(ifsc_name,bank_name,city_name)
